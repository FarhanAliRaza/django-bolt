# Django-Bolt

High-performance API framework for Django with Rust-powered endpoints delivering 60k+ RPS.

## Quick Start

```bash
pip install django-bolt
```

## Features

- **Rust-Powered Performance**: 60k+ requests per second
- **Django Integration**: Works with existing Django projects
- **Type Safety**: Full type hints and validation with msgspec
- **Async Support**: Native async/await support
- **OpenAPI**: Auto-generated API documentation
